Detailed Description

template<size_type I, typename T>
struct wlp::TupleElement< I, T >

Tuple is a flat implementation so that compile-time template instantiations are not recursive. The tuple implementation will inherit from an element type that contains the type index and the held type. The index is needed to discern multiple types.

Template Parameters
Ithe element index
Tthe tuple element type
