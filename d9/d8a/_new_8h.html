Detailed Description

New operators defined for Arduino boards and for other boards use the normal new.

Some versions of Arduino provide a

new.h

but these do not include a definition for placement new. In debug mode, for Wlib develop, the standard C++

<new>

is included, and for all other cases, these operators fall back on the C allocation functions.
