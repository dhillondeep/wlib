template<typename Key , typename Val , typename Hasher , typename Equals >
void wlp::ChainHashMap< Key, Val, Hasher, Equals >::ensure_capacity ( )
private

Resize and rehash the hash map if the current load factor exceeds or equals the maximum load factor. This function will double the size of the backing array, allocating a new array, and fully deallocating the previous array.

Precondition
this function will create a new array allocator, however the same node allocator will be used, which means that the node allocator will start drawing from heap memory if the number of elements exceeds the initial max elements
