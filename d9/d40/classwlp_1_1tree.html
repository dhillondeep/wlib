Detailed Description

template<typename Element, typename Key, typename Val, typename GetKey, typename GetVal, typename Cmp = comparator <Key>>
class wlp::tree< Element, Key, Val, GetKey, GetVal, Cmp >

Red black tree implementation, designed for use in associative container implementation, e.g.

Map

and

Set

. The insertion and deletion operations are based on those in CLRS.

The provided

Element

type must provide functions

get_key()

which returns a type of

Key

and a function

get_val()

which returns a type of

Val

. These details are dealt by implementing data structures such as

Map

and

Set

.

Template Parameters
Elementelement type which contains the key and val
Keynode key type
Valnode value type
Cmpkey comparator type, which uses the default comparator
GetKeyfunctor type used to get element key
GetValfunctor type used to get element value
