template<typename Key, typename Val, typename Hasher = Hash<Key, uint16_t>, typename Equals = Equal<Key>>
class wlp::HashMap< Key, Val, Hasher, Equals >

Hash map implemented using separate chaining, in the spirit of std::unordered_map.
