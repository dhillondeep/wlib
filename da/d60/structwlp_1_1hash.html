Detailed Description

template<class Key, class IntType>
struct wlp::hash< Key, IntType >

A basic hash function is defined to be a function that, as best as reasonably possible, maps a key-type value to a unique positive integer value.

Precondition
The default hash function attempts to cast to the integer type, which works for integer, character, and floating point types.
