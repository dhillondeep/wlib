Detailed Description

template<typename Key, typename Val, typename Hasher = hash<Key, uint16_t>, typename Equals = equals<Key>>
class wlp::open_map< Key, Val, Hasher, Equals >

Hash map implemented using open addressing and linear probing, in the spirit of std::unordered_map.
