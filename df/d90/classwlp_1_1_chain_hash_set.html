Detailed Description

template<class Key, class Hash = Hash<Key, uint16_t>, class Equal = Equal<Key>>
class wlp::ChainHashSet< Key, Hash, Equal >

Hash set implementation using separate chaining. This implementation supports removal operations, unlike the open addressed set.

Template Parameters
Keythe element type
Hashthe hash function
Equalthe equality function
