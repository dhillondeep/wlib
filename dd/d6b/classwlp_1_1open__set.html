Detailed Description

template<class Key, class Hasher = hash <Key, uint16_t>, class Equals = equals <Key>>
class wlp::open_set< Key, Hasher, Equals >

An open hash set is created using a backing hash map, and all available functions are a subset of the functions of the hash map. The set contains unique elements.

Template Parameters
Keythe unique element type
Hashthe hash function of the stored elements
Equaltest for equality function of the stored elements
