template<typename Key, typename Val, typename Ref, typename Ptr, typename Hasher, typename Equals>
struct wlp::ChainHashMapIterator< Key, Val, Ref, Ptr, Hasher, Equals >

Iterator class over the elements of a ChainHashMap. Specifically, this class iterates through each chain and then the backing array.
