Detailed Description

template<typename Key, typename Val, typename Cmp = Comparator <Key>>
class wlp::RedBlackTree< Key, Val, Cmp >

Red black tree implementation, designed for use in associative container implementation, e.g.

Map

and

Set

. The insertion and deletion operations are based on those in CLRS.

Template Parameters
Keynode key type
Valnode value type
Cmpkey comparator type, which uses the default comparator
