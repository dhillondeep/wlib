Detailed Description

template<typename Key, typename Val, typename Hasher = Hash<Key, uint16_t>, typename Equals = Equal<Key>>
class wlp::OpenHashMap< Key, Val, Hasher, Equals >

Hash map implemented using open addressing and linear probing, in the spirit of std::unordered_map.
