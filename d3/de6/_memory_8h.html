Detailed Description

Memory is a class that provides dynamic fixed size memory.

It does not impose any restrictions on how many blocks can be borrowed and of what size can it be borrowed. This has an option to use a memory pool which has restrictions on how much memory we can use. When pool is filled, any call for memory will return

nullptr
Precondition
Number of memory block sizes will depend on
MAX_ALLOCATORS

Deep Dhillon
